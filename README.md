# 初中英语7年级下册学习平台

## 项目简介
一个面向初中7年级学生的英语学习Web应用，包含单词学习、语法练习、题库训练、智能复习等16个功能模块，基于SM-2间隔重复算法实现科学记忆。

## 功能列表

### 核心学习模块 (6个)
| 模块 | 说明 | 数据 |
|------|------|------|
| 智能学习 | SM-2算法 · 7模式 · 单元解锁 | 190词 |
| 单词学习 | 6单元 · 分单元浏览 | 190词 |
| 语法学习 | 30要点 · 图解 · 例句 | 30点 |
| 高级练习 | 综合训练 · 难点突破 | - |
| 题库练习 | 选择/填空/完形 | 50题 |
| 期末备考 | 模拟测试 · 查漏补缺 | - |

### 专项训练模块 (4个)
| 模块 | 说明 |
|------|------|
| 错题本 | 自动收集 · 错因分析 · 重练 |
| 增强学习 | 词根词缀 · 派生词 · 联想 |
| 拼写练习 | 听写·默写·逐字母输入 |
| 单词翻翻卡 | 3D翻转 · 快速记忆 |

### 学习工具模块 (6个)
| 模块 | 说明 |
|------|------|
| 学习进度 | 可视化图表 · 数据统计 |
| 复习计划 | 艾宾浩斯 · 今日待复习 |
| 成就系统 | 20成就 · 积分奖励 |
| 学习目标 | 每日目标 · 打卡激励 |
| 学习助手 | AI问答 · 学习建议 |
| 音效设置 | 背景音乐 · 音量调节 |

## 快速启动
### 方法一：使用启动脚本
1. 双击 `启动应用.bat`
2. 浏览器打开 http://localhost:8080

### 方法二：手动启动
```bash
cd english-learning-app
node server.js
```

## 技术栈
- **前端**: HTML5 + CSS3 + Vanilla JavaScript (ES6+)
- **存储**: localStorage (浏览器本地)
- **服务器**: Node.js (server.js)
- **算法**: SM-2 间隔重复算法
- **语音**: Web Speech API (TTS)

## 数据概览
- 单词: 190个 (6个单元)
- 语法点: 30个

- 成就: 20个
- 每日挑战: 4个

## 目标用户
- **对象**: 初中7年级学生
- **场景**: 课后自学、考前复习、日常积累
- **指标**: 学习效率提升3倍（SM-2算法）、正确率>80%
